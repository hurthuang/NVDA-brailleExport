# NVDA-brailleExport
# 點字輸出匯出 (Braille Export)

這是一個適用於 NVDA 螢幕報讀軟體的附加元件 (Global Plugin)。它的主要功能是攔截 NVDA 傳送至點字顯示器的輸出內容，並將其匯出成電腦檔案或直接複製到剪貼簿。這對於需要製作點字教材、記錄點字顯示畫面、或是將點字內容送交點字印表機 (Embosser) 的使用者非常方便。

## ✨ 主要特色

* **靈活的輸出目標**：可選擇將點字內容直接**複製到剪貼簿**（預設），方便隨時貼上；或選擇**儲存成實體檔案**。
* **畫面快照 (Snapshot)**：一鍵擷取目前點字顯示器上的內容並輸出。
* **連續錄製 (Recording)**：記錄一段時間內的點字畫面變化，適合用來錄製連續閱讀或操作的點字軌跡。
* **多種文字格式**：
  * **Unicode 點字文字 (`.txt` / 剪貼簿)**：保留完整的 8 點點字字元（例如 `⠓⠑⠇⠇⠕`），適合在電腦上視覺預覽與保存。
  * **BRF ASCII 點字 (`.brf` / 剪貼簿)**：採用標準北美點字電腦碼 (NABCC)，過濾為標準的 6 點點字與半形空白，可直接送交點字壓印機印出或於專業點字軟體中編輯。

## 🚀 系統需求
* NVDA 螢幕報讀軟體
* 搭配支援的點字顯示器使用（或開啟 NVDA 點字檢視器）

## ⌨️ 預設快捷鍵與操作

你可以在 NVDA 的 **「輸入手勢」** 對話框中（分類：`點字輸出匯出`）自訂或修改以下指令：

* `NVDA + Shift + C`：**立即匯出目前點字顯示畫面快照**
* `NVDA + Shift + R`：**開始錄製點字輸出**（再按一次即可停止並輸出）
* *(無預設)*：開啟點字匯出設定對話框

此外，你也可以透過 **NVDA 系統匣選單 (右下角圖示右鍵)** -> **[點字匯出 (X)]** 來執行上述功能與開啟設定。

## ⚙️ 設定選項說明

進入 **[設定 (O)...]** 後，你可以調整以下項目：

1. **輸出目標**：
   * **複製到剪貼簿 (預設)**：將快照或錄製好的點字文字直接存入系統剪貼簿，方便你貼到任何文件或通訊軟體中。
   * **儲存成檔案**：將內容輸出為實體的 `.txt` 或 `.brf` 檔案。
2. **匯出目錄**：*(僅在儲存成檔案時有效)* 設定檔案要儲存的資料夾路徑，預設為使用者的「桌面」。
3. **文字格式**：選擇要使用 Unicode 點字還是 BRF ASCII 點字編碼。
4. **檔名加入時間戳記 (T)**：*(僅在儲存成檔案時有效)* 開啟後，每次匯出的檔名都會自動加上目前的系統時間（例如：`braille_record_20260403_120000.txt`），避免檔案被覆蓋。
5. **每行格數 (C)**：輸出的自動換行設定。若你的點字印表機紙張或文件排版每行最多 40 格，可設定為 40，內容會自動在此長度進行換行。
